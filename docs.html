Canvas API Documentation

A real-time pixel canvas API

GET /getPixel

Get the color of a pixel at specific coordinates.

GET /getPixel?x=100&y=100

POST /updatePixel

Update the color of a pixel at specific coordinates.

POST /updatePixel
{
    "x": 100,
    "y": 100,
    "r": 255,
    "g": 0,
    "b": 0
}

WebSocket /ws

Connect to receive real-time canvas updates.

WebSocket /cws

Connect to send drawing commands.
